ILUSTRE-PROJECT HD-2D

Platforms: Windows
Game Engine: Unreal Engine 5
Team Size: 4
Roles: Programmer and Music composer
Non commercial Project
Synopsis: Summary of contributions:
Ilustre is a cooperative multiplayer game with HD-2D style graphics in which at the beginning of the game you can choose to be a warrior, an archer, a wizard or a guardian.
Each class has different skills that will help you solve different puzzles found in the world, so you will have to get together with your friends and coordinate to overcome the challenges of this game.
  • Player movement and skills
  • Player animations using PaperZD plugin
  • Implementation of multiplayer
  • UI implementation
  • Heal and Damage system
  • Enemy AI
