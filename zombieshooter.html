ZOMBIE SHOOTER PROJECT

Platforms: PC
Game Engine: Unity
Team Size: 1
Non commercial Project
Synopsis:

This project was originally intended to be VR, so it has some VR features.

For testing I implemented a classic first person system because I didn't have a VR system available at times during development.

It has some similarities with Call of Duty zombies, such as the economy, the rounds system and that the zombies follow you all the time no matter the distance.
Features:
  • Health bar
  • Earn points by eliminating zombies
  • Possibility to switch to VR (Simulator)
  • Grab System in VR
  • Rounds System
